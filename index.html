Phase 1 MVP

Daily Utility App

A lightweight Android-first app built with Python + Kivy for daily notes and reminders.

See Features How to Run

Current Features (Phase 1)

  • Notes: add, view, delete
  • Reminders: add, view, delete
  • Reminder categories: Study, Workout, Tasks
  • Repeat options: Once, Daily, Weekly
  • Local SQLite storage (offline)
  • Bottom navigation UI (Home / Notes / Reminders / Assistant)

How to Run the App

This repository hosts source code. The mobile app itself runs locally in Pydroid 3 / Termux or as an APK built with Buildozer.

  1. Install Kivy in your runtime environment.
  2. Run python main.py for local testing.
  3. Or build APK with buildozer android debug.
